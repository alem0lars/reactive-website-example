    README for Higgs Reports

    Description

    Higgs reports collect the build informations for the projects that uses Higgs as continuous integration server.

    Users

    There are 2 kind of users:

    • *Admin Users*: They are the Higgs administrators. This user kind grants the privileges to manage all Higgs aspects.

    • Users: They are normal Higgs users. They can visit the website, or create new projects, based on their role.

    The available roles for Users are:

    • Visitor: It can visit all public projects.

    • Manager: It can manage projects that it owns. A project owner is the User that creates the project. For now, the project owner can be changed only by admin users.
